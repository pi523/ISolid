## 是什么

你有没有在洗澡、通勤、睡前突然冒出一个想法——然后因为"等一下再想"而彻底放一边了，或者有了想法之后不知道怎么推进，渐渐的就遗忘了？

ISolid 就是解决这个问题的。输入你的想法（哪怕只有一句话），AI 会帮你：

1. **追问关键细节**——逐题问清楚时间线、目标用户、核心功能、最大顾虑
2. **生成精简预览**——一份你能快速确认或修改的结构化摘要
3. **写出完整 PRD**——11 个维度的产品需求文档，含竞品分析
4. **拆出 GTD 行动清单**——按情境分类（@computer @phone @errand……）
5. **持续跟进**——根据你的热情程度定期问询进度，不让想法烂在笔记里

---

## 核心流程

```
输入想法（一句话到几段话都行），毫无结构梦到哪里写哪里就行
    ↓
AI 动态追问（每轮 5 题，多选 + 自由填写）
    ↓
确认精简预览 · 选热情程度（L1–L4）
    ↓
自动生成 PRD 详细版 + GTD Todo
    ↓
定期进度问询 · 回复更新状态
```

## 热情程度

| 等级 | 含义 | 跟进频率 |
|---|---|---|
| L1 一时兴起 | 记录即可，不催 | 月度回顾 |
| L2 有点意思 | 轻提醒 | 两周一次 |
| L3 想落地·缓和 | 周度跟进 | 每周 |
| L4 想落地·迫切 | 高频推进 | 约每 2.5 天 |
